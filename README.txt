Project: WeatherKids Website

Uses: MySQL database, HTML, PHP (7.2); WampServer 3

Information: Project contains html and imbeded PHP code to present a functional
weather information website for kids to join and sign up for activities. php 
accesses mysql database through use of WampServer and apache to display information
on local host machine. Database screenshot is included to assist in the recreation of
the table used in the code, in order for replication of the website and all its 
related features.
